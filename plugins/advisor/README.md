# Advisor

Advisor is a Codex-only plugin that packages the local `$advisor` skill and its read-only `advisor` agent for marketplace installation.

Use it when planning work is still fuzzy: product/design/engineering tradeoffs, scope cuts, spec shaping, ticket planning, and "challenge this" requests. The skill delegates that work to a read-only advisor subagent so the main Codex thread can stay focused on the user's next decision or implementation step.

Claude Code is intentionally not supported by this plugin because Claude Code already has built-in advisor mode. Do not add this plugin to the Claude Code marketplace or Claude behavior evals.
